# Installation

## 1. 下载
```
git clone https://github.com/pandoraxm/laravel-react-curd.git
```


## 2. composer安装

```
cd laravel-react-curd

composer install
```

## 3. 修改env以及数据库配置

```
cp .env.example .env

vim .env
```

## 4. 生成key

```
php artisan key:generate

```

## 5. 迁移

```
php artisan migrate
```



## 6. npm安装

```
npm install

npm run dev
```
